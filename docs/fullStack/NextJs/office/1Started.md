# 第一章 开始

## 1.1 安装

```bash
npm install -g  pnpm # 安装pnpm
npx create-next-app@latest nextjs-dashboard --example "https://github.com/vercel/next-learn/tree/main/dashboard/starter-example" --use-pnpm # 创建项目

cd nextjs-dashboard
pnpm i
pnpm dev # 启动项目

```
