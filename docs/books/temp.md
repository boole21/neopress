# 电商计划

## 1 步骤

- 定位：医护类如护士服，被套等
- 选品: 质量，款式等
- 营销：seo，youtube广告
- 采购: 1688，拼多多等
- 运输：云途
- 仓库: 美国或中国直接发货
- 发货: 美国发货
- 售后: 回复邮件

## 2 优势

美国医院统一服装，需要去美国线下跑业务,国内的做跨境电商的无法做到.
对服装材料，制造质量把控

## 3 投入

1. 业余时间
2. 中国采购商品，以一件衣服50元算,10件500元
3. 物流，邮政7-30天,0.1kg13.7元,1kg算137元
4. 美国发货或中国一件发货
5. 建站，优化，广告推广,订单处理
6. 售后，退换货
